## 📊 主要発見

### 1. エグジット理由全体分布（11,013件）
- **損切**: 40.2%（最多、リスク管理として機能）
- **デッドクロス**: 38.4%（2番目、トレンドフォロー型として機能）
- **利益確定**: 12.1%
- **トレーリングストップ**: 8.0%（限定的だが大きな利益獲得に寄与）
- **強制決済**: 1.3%

### 2. 仮説1検証結果: **不成立**
**仮説**: 損切3%5%7%で差がない理由 → トレーリングストップが多い

**検証結果**:
- トレーリングストップ比率: 8.0%（多くない）
- 損切幅別PF標準偏差: 0.0000（完全一致）
- 真の理由: デッドクロス38.4%とトレーリング8.0%が損切幅に関わらず一定比率

**結論**: トレーリングストップが多いのではなく、「デッドクロスが支配的」なため損切幅の違いが意味を持たない。

### 3. 仮説2検証結果: **成立**
**仮説**: トレーリング20%25%30%でペイオフレシオ改善

**検証結果**:
- トレーリング5%: ペイオフレシオ1.87
- トレーリング20-30%: ペイオフレシオ2.11
- 改善率: +12.9%（5%以上の有意な改善）

**結論**: トレーリング幅拡大でペイオフレシオ改善。ただし、**トレーリング10%が最適**（ペイオフ2.15、PF1.15）。

### 4. トレーリングストップ発動率の真実
- **5%**: 発動率39.3%（過剰、小さな利益で頻繁に利確）
- **10%**: 発動率4.7%（適切、損切・デッドクロスとバランス）
- **15%以上**: 発動率0-0.2%（ほぼ発動せず、損切・デッドクロスが優先）

**示唆**: トレーリング幅は「発動頻度」ではなく「ペイオフレシオ最大化」で選ぶべき。
