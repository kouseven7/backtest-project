### 普遍性スコア: 0.10

**解釈**: 武田薬品のみ劇的改善（+157.2%）。他9銘柄では全て悪化または効果なし。

### Phase 1.6発見3の検証結果

**武田薬品ATRパラドックス**:
- **超高ATR（4%+）**: 勝率100%、平均利益13.70%
- **低ATR（0-2%）**: 勝率22.1%、平均損益率-1.38%

**今回の検証**:
- 低ATR除外前: PF=0.53
- 低ATR除外後: PF=1.36（+157.2%改善）
- 除外された低ATR取引: PF=0.40、勝率22.1%（Phase 1.6と完全一致）

**結論**: ATRパラドックスは**武田薬品特有の現象**であり、他銘柄には適用できない。

### なぜ他銘柄で悪化したのか？

**仮説**: 低ATR＝値動きが小さい＝ストップロスに引っかかりやすいという前提が誤り

**実データ分析**:
- **6501.T**: 低ATR（<2.0%）のPF=3.43、勝率57.4%（**高成績**）
- **9983.T**: 低ATR（<2.0%）のPF=8.65、勝率63.3%（**極めて高成績**）
- **6758.T**: 低ATR（<2.0%）のPF=2.00、勝率43.8%（**高成績**）

**逆説的発見**: 多くの銘柄で**低ATRの方が高PF・高勝率**という結果が得られた。

**メカニズム推定**:
1. 低ATR時期は市場が安定している
2. 安定期は緩やかなトレンドが持続しやすい
3. トレンドフォロー戦略にとって有利

**武田薬品の特殊性**:
- 急騰→急落パターン（56.1%の取引）が支配的
- 高ATR時のみトレンドが持続する特異な値動き

## 3. ATR利確ルール（実装検討）

**設計**: [PHASE1_9_FILTERS_DESIGN.md](PHASE1_9_FILTERS_DESIGN.md) フィルター3

### ロジック概要

```python
利確価格 = entry_price × (1 + entry_atr_pct / 100 × 10)
```

**例**:
- エントリー価格: 1000円
- ATR: 2.0%
- 利確価格: 1000 × (1 + 2.0 / 100 × 10) = 1000 × 1.20 = 1200円（+20%）

### 実装課題

**問題**: 既存のCSVデータ（Phase 1.6グリッドサーチ結果）は全て**エグジット済み**のため、新しい利確ルールを適用できない。

**解決策**:
1. **BaseStrategy統合**: `strategies/base_strategy.py`に`atr_profit_target`パラメータを追加
2. **バックテスト再実行**: 10銘柄で新しい利確ルールを適用したバックテストを実行
3. **結果比較**: 既存の利確ルール vs ATR利確ルール

### 実装スケジュール

## 4. 複合フィルター設計（Phase 1.10への提言）

### 単体フィルター評価サマリー

| フィルター | 普遍性スコア | 推奨度 | 理由 |
|-----------|-------------|-------|------|
| トレンド強度（高のみ） | 0.33 | ⭐⭐⭐ | Phase 1.8で検証済み、部分的に有効 |
| MA乖離<5% | 0.40 | ⭐⭐⭐⭐ | 最も普遍性が高い |
| ATR範囲（中〜高のみ） | 0.10 | ⭐ | 武田特有、他銘柄では逆効果 |

### 複合フィルター候補（AND条件）

#### 候補1: トレンド強度 AND MA乖離（推奨度: 高）

```python
entry_ok = (entry_trend_strength >= threshold_high) and (sma_distance_pct < 5.0)
```

**期待効果**:
- トレンド強度フィルター: 3/9銘柄で改善（Phase 1.8）
- MA乖離フィルター: 4/10銘柄で改善（Phase 1.9-A）
- **理論上の改善銘柄**: 両方で改善した銘柄（共通集合）

**検証すべき銘柄**:
- 8306.T（三菱UFJ）: トレンド+90.5%、MA乖離+26.0% → 両方で改善
- 4063.T（信越化学）: トレンド+41.6%、MA乖離-18.0% → トレンドのみ改善

#### 候補2: トレンド強度 AND MA乖離 AND ATR範囲（推奨度: 中）

**懸念**: ATR範囲フィルターが逆効果の可能性が高いため、慎重に検証すべき。

#### 候補3: MA乖離のみ厳格化（推奨度: 中）

```python
entry_ok = sma_distance_pct < 3.0  # 5.0%から3.0%に厳格化
```

**期待効果**: 取引数をさらに削減し、PFを向上させる可能性。

### 複合フィルター候補（OR条件）

#### 候補4: トレンド強度 OR MA乖離（推奨度: 中）

```python
entry_ok = (entry_trend_strength >= threshold_high) or (sma_distance_pct < 5.0)
```

**期待効果**: 取引数の減少を抑えつつ、ノイズを除去する緩和モード。

**結論**: 部分的に有効。複合フィルターで強化すべき。

### 発見3: ATRパラドックス

**Phase 1.6**: 武田薬品で超高ATR勝率100%、低ATR勝率22.1%

**Phase 1.9-B**: 武田のみ+157.2%改善、他9銘柄では逆効果（低ATRの方が高PF）

**結論**: 武田薬品特有。他銘柄への適用は不可。

### 発見4: SMA乖離5%以上での全敗

**Phase 1.6**: 武田薬品で33件全敗

**結論**: **普遍的に有効**。複合フィルターの基盤とすべき。
