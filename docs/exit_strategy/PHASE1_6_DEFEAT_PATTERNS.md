## 📊 エグゼクティブサマリー

### 主要発見
1. **大敗パターン**: 18件 / 180件（**10.0%**）
2. **大敗銘柄**: **武田薬品工業（4502.T）のみ**が全18パラメータで大敗
3. **除外推奨パラメータ**: **なし**（大敗はパラメータではなく銘柄依存）
4. **残存データ**: 162件（90.0%）が大敗基準をクリア

### 結論
- **パラメータ起因の大敗なし**: トレーリング拡張（20%, 25%, 30%）は有効
- **銘柄起因の大敗**: 武田薬品（4502.T）は戦略と相性悪い→除外検討
- **次ステップ**: Task 5（ペイオフレシオ分析）で残存162件を分析可能

## 🎯 大敗の定義（OR条件）

以下のいずれか1つでも該当すれば「大敗」と判定：

| 基準 | 閾値 | 理由 |
|------|------|------|
| **PF** | < 0.7 | Phase 1.5最低値0.69を参考 |
| **最大ドローダウン率** | > 30% | 資金管理上の許容限界 |
| **単一銘柄の最大損失** | > 15% | 1銘柄の壊滅的損失を回避 ※本分析では未実装 |

**注**: 単一損失列が存在しないため、PF < 0.7とDD率 > 30%のみで評価。

### 4. 除外推奨パラメータ

**結論**: **除外推奨パラメータなし**

- 大敗率 > 50%のパラメータ組み合わせ: **0組**
- 各パラメータの大敗率: 10%（10銘柄中1銘柄のみ失敗）
- トレーリング拡張（20%, 25%, 30%）は有効と判断

**根拠**:
```
各パラメータの大敗率 = 1銘柄（武田薬品） / 10銘柄 = 10%
→ 50%閾値を大幅に下回る
→ パラメータではなく銘柄の問題
```

## 💡 考察とインサイト

### 1. トレーリング拡張の有効性
- **Phase 1.5（トレーリング5%/10%/15%）**: 大敗パターンあり（武田薬品）
- **Phase 1.6（トレーリング20%/25%/30%追加）**: 大敗パターン同数（武田薬品のみ）
- **結論**: トレーリング拡張により**新たな大敗は発生しなかった**

### 2. 武田薬品の異常性
- **全パラメータで失敗**: パラメータ調整では改善不可能
- **PF 0.479（固定）**: 損失が利益の2.1倍
- **DD率 45.7%**: 資金管理上の許容限界（30%）を大幅超過
- **推測原因**:
  - ゴールデンクロス頻発後の即反転（ダマシ多発）
  - トレンド形成が弱い（薬品株特有の値動き？）
  - 2020-2025年の6年間で戦略と相性最悪

### 3. 残り9銘柄の健全性
- **全銘柄が大敗回避**: PF 0.7以上、DD率 30%以下を維持
- **業種分散が有効**: 自動車・通信・銀行・電機・小売・商社・化学
- **9銘柄×18パラメータ = 162件**がTask 5分析対象
