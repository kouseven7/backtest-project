### 変更ファイル

### 変更詳細
```python

"filter_mode": "or",

# 変更後（AND条件）
```

- **過剰取引削減**: 63.6%の取引削減（品質向上トレードオフ）
- **条件厳格化**: ADX >= 67%ile **かつ** SMA乖離 < 5.0%の両条件満たす場合のみエントリー
