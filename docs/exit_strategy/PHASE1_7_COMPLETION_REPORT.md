## 結論と次ステップ

### Phase 1.7の結論

1. **武田薬品の大敗パターンは普遍的でない**
   - 5つの発見事項のうち、普遍性スコア >= 0.6 の項目なし
   - 最高スコア: 発見4（SMA乖離5%以上）= 0.33（3/9銘柄）

2. **適正パラメータの特定**
   - 損切: **3%**（最良）
   - トレーリング: **10-30%**（幅広い範囲で有効）
   - 利確: **なし**（シンプル維持）

3. **適正PF範囲の確立**
   - **推奨: 1.2-2.0**（9銘柄の実績ベース）
   - カーブフィッティング回避: PF > 3.0は警戒

### Phase 1.8への提案

#### 選択肢A: フィルター実装（推奨せず）

**理由**: 普遍性スコアが低い（最高0.33）ため、フィルター実装の効果は限定的。

**もし実装する場合の優先度**:
1. 発見4: SMA乖離5%以上でのエントリー禁止（スコア0.33）
2. 発見5: 極端下落相場の検出（スコア0.22）
3. 発見3: ATRベースの利確（スコア0.11）

#### 選択肢B: パラメータ確定（推奨）

**Phase 1.6で特定された最良パラメータを確定**:

```python
FINAL_EXIT_PARAMS = {
    'stop_loss_pct': 0.03,          # 損切3%（TOP 1）
    'trailing_stop_pct': 0.10,      # トレーリング10%（TOP 1）
    'take_profit_pct': None         # 利確なし（シンプル維持）
}
```

**期待される成績**:
- 平均PF: 1.23（9銘柄平均）
- 平均勝率: 35.9%
- 合格率: 66.7%（6/9銘柄でPF > 1.0）
- カーブフィッティングリスク: 低（複数銘柄で検証済み）

#### 選択肢C: Out-of-Sample検証（最推奨）

**Phase 2への移行**:
- In-Sample期間（2020-2023）で選定したパラメータを、Out-of-Sample期間（2024-2025）で検証
- 劣化率 < 30%を確認
- 汎化性能の最終確認

## 重要な教訓

### カーブフィッティング回避

1. **PF 13は異常値**
   - 武田薬品での極端なPF改善（0.53→13.96、+2538%）は実トレードで再現不可能
   - Phase 3-6でPF=121.07が-98.9%崩壊した過去の失敗を再現しかねない

2. **適正PF範囲の遵守**
   - 推奨: 1.2-2.0（9銘柄の実績ベース）
   - 上限: 3.0（セーフティマージン）

3. **複数銘柄検証の重要性**
   - 単一銘柄最適化は過学習の温床
   - 9銘柄で66.7%合格は妥当な汎化性能

### Phase 1.6グリッドサーチの有効性

- **18組み合わせ × 9銘柄 = 162データポイント**で現実的なパラメータ空間を網羅
- 合格率66.7%（6/9銘柄）は実トレードでの成功確率として妥当
- トレーリング10-30%の幅広い範囲で有効性確認
