# ペイオフレシオ・エグジット分析プロジェクト

**作成日**: 2026年1月23日  
**目的**: 大敗しないエグジット戦略の可能性を残し、PF1.2以上達成  
**実施者**: Backtest Project Team  
**プロジェクト方式**: タスク管理型（独立プロジェクト）

---

## 📋 プロジェクト目的

### 主目的
大敗しないエグジット戦略を目指すため、大負けしないパターンと大負けするパターンを洗い出して分類し、大敗のないパターンを積み上げてPF2が可能なパラメータを残す。

### 背景
- **Phase 1失敗**: 平均PF=1.15、合格率60%（目標80%未達）
- **Phase 1.5失敗**: 全9組み合わせで合格率60%止まり
- **根本原因**: Win Rate 34.7%（目標40%未達）
- **新方針**: エントリー改善よりエグジット最適化にフォーカス

### 除外すべき未来情報パターン
**例1**: シグナル発生日の終値でエントリー
- 結果: PF100以上の異常値
- 理由: 未来情報混入、実データで破綻
- **判定**: 除外データに列挙

**例2**: シグナル発生日の翌日始値でエントリー/エグジット（採用パターン）
- 結果: PF 0.7-1.88（正常範囲）
- 理由: 未来情報なし、実運用可能
- **判定**: 日次データ取引を前提条件とする

## 📊 大敗の定義

以下のいずれかを満たす場合を「大敗」と定義：

| 基準 | 閾値 | 理由 |
|------|------|------|
| **PF** | < 0.7 | Phase 1.5最低値0.69を参考 |
| **最大ドローダウン** | > 30% | 資金管理上の許容限界 |
| **単一銘柄の最大損失** | > 15% | 1銘柄の壊滅的損失を回避 |

**大敗パターンの扱い**:
- 該当パラメータは最終推奨から除外
- 大敗理由を分析・ドキュメント化
- 除外パターンリストを作成

### Task 6: PF2達成パラメータ選定
**目的**: PF2に最も近いパラメータを推奨  （大敗はまだ除外しない）
**期間**: 1時間  
**成果物**: 
- 最終推奨パラメータ（TOP 3）
- `docs/exit_strategy/PHASE1_6_FINAL_RECOMMENDATIONS.md`

**選定基準**:
```python
# 大敗除外
non_defeat = results_df[
    (results_df['profit_factor'] >= 0.7) &
    (results_df['max_drawdown'] <= 0.30) &
    (results_df['max_single_loss'] <= 0.15)
]

# 銘柄平均でグループ化
grouped = non_defeat.groupby(['stop_loss_pct', 'trailing_stop_pct']).agg({
    'profit_factor': 'mean',
    'payoff_ratio': 'mean',
    'win_rate': 'mean'
}).reset_index()

# 複合スコア
grouped['score'] = (
    grouped['profit_factor'] * 0.5 +       # PF重視
    grouped['payoff_ratio'] * 0.3 +        # ペイオフレシオ
    grouped['win_rate'] * 100 * 0.2        # Win Rate
)

# TOP 3選定
top3 = grouped.nlargest(3, 'score')
```

**成功基準**:
- [ ] 大敗除外後のパラメータ数 > 0
- [ ] TOP 3パラメータのPF ≥ 1.3
- [ ] TOP 3パラメータのペイオフレシオ ≥ 1.8
- [ ] 最終推奨レポート作成
